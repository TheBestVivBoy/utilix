Setup & Commands

🎫 Ticket Commands

/createembed

Create a ticket panel for users to open tickets.

/removeembed

Remove an existing ticket panel.

/add

Add a user to the ticket.

/remove

Remove a user from the ticket.

/removeall

Remove all added users except the owner.

/close

Close the current ticket.

/closerequest

Request ticket closure from the owner.

/blacklist

Block a user from creating tickets.

/unblacklist

Unblock a user and allow ticket creation again.

/listembeds

List all ticket panels.

/listtickets

List all active tickets.

/ticketspurge

Purge all tickets (optionally by category).

/ticketstats

View stats for a ticket channel.

🛡️ Moderation Commands

/kick

Kick a user from the server.

/ban

Ban a user from the server.

/tempban

Temporarily ban a user.

/warn

Warn a user for breaking rules.

/unwarn

Remove a warning from a user.

/unban

Unban a previously banned user.

/unmute

Unmute a user in the server.

/history

View a user’s moderation history.

/resetwarnings

Reset all warnings for all users.

/purgehistory

Reset moderation history for a specific user.

/user

View detailed information about a user.

🕵️ Scam Database

/check

Check if a user or server is reported.

/report

Report a user or server with proof.

🛠️ Utilities & 🌐 Tools

/sticky

Manage sticky messages in channels.

/timestamp

Convert natural language to Discord timestamps.

/settime

Set a user’s timezone.

/time

View a user’s local time.

/translate

Translate text between languages.

/purgeinvites

Remove all server invite links from messages.
